```

```

那么，经过顺时针转90度的矩阵为

```

32 22 12
33 23 13
```

| ---- | -------- | ---- | ---- | ----------- | ---- | ---- |
| 3    |          | 1    | 1    |             | 1    | 3    |
| 3    |          | 1    | 2    |             | 2    | 3    |
| 3    |          | 1    | 3    |             | 3    | 3    |

| 3    |          | 2    | 2    |             | 2    | 2    |
| 3    |          | 2    | 3    |             | 3    | 2    |

| 3    |          | 3    | 2    |             | 2    | 1    |
| 3    |          | 3    | 3    |             | 3    | 1    |
