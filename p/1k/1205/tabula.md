设原矩阵为

```
11 12 13
21 22 23
31 32 33
```

那么，经过顺时针转90度的矩阵为

```
31 21 11
32 22 12
33 23 13
```



| n    | original | x    | y    | Transformed | tx   | ty   |
| ---- | -------- | ---- | ---- | ----------- | ---- | ---- |
| 3    |          | 1    | 1    |             | 1    | 3    |
| 3    |          | 1    | 2    |             | 2    | 3    |
| 3    |          | 1    | 3    |             | 3    | 3    |
|      |          |      |      |             |      |      |
| 3    |          | 2    | 1    |             | 1    | 2    |
| 3    |          | 2    | 2    |             | 2    | 2    |
| 3    |          | 2    | 3    |             | 3    | 2    |
|      |          |      |      |             |      |      |
| 3    |          | 3    | 1    |             | 1    | 1    |
| 3    |          | 3    | 2    |             | 2    | 1    |
| 3    |          | 3    | 3    |             | 3    | 1    |

可以推出:
$$
t[y][n-x+1]==o[x][y]
$$
